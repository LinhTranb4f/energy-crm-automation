# HubSpot ↔ n8n Integration Map

This document maps **HubSpot workflows** that trigger **n8n processes** via webhooks, so you can quickly see which n8n workflow runs when a given HubSpot workflow fires. Use it to review and trace the integration between HubSpot and n8n.

**Sources:** [HubSpot workflows](../hubspot/README.md) · [n8n workflows](../n8n/README.md)

---

## How to use this map

- **From HubSpot:** Find your workflow ID (e.g. 2051) in the tables below → see the n8n webhook path and the n8n doc to open.
- **From n8n:** Find the webhook path (e.g. `push-service-to-Delfin`) → see which HubSpot workflow(s) send to it and the n8n workflow doc.

**Webhook base URL (production):** `https://n8n.tools.energy/webhook/`  
*(Exception: workflow 1016 uses `https://energy-n8n.business4you.ch/webhook/`.)*
