### Overview

This workflow is designed to automatically trigger an external commission generation process via a webhook. It activates when financial records (specifically Invoices or Credit Notes) meet certain criteria. The workflow is currently inactive.

### Enrollment Triggers

A custom object record will be enrolled in this workflow if it meets **EITHER** of the following conditions:

*   **Condition 1: New Invoice**
    *   The record's **Record Type** is 'Invoice'.
    *   AND the record's **Invoice Stage** is updated to 'Create invoice'.
*   **Condition 2: Credit Note with Commission**
    *   The record's **Record Type** is 'Credit Note'.
    *   AND the record's **Commission Amount** property has any value (is known).

### Re-enrollment

*   Re-enrollment is enabled for this workflow.
*   A record will be re-enrolled each time its **Commission Amount** property is updated with a new value.

### Actions

When a record is enrolled, the workflow will immediately perform the following action:

*   **Action: Send a Webhook**
    *   **Purpose:** To trigger an automated process in an external system (n8n) to generate a commission.
    *   **Method:** POST
    *   **URL:** [https://n8n.tools.energy/webhook/generate-commission](https://n8n.tools.energy/webhook/generate-commission)
    *   **Authentication:** Uses a secret 'Authorization' key sent in the request header.
